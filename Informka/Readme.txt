Как работать с данным документом
--------------------------------

При просмотре файла README.txt на экране с помощью текстового редактора 
Блокнот разверните окно редактора до максимального размера.

ПЕРЕЧЕНЬ ФАЙЛОВ С ДОПОЛНИТЕЛЬНЫМИ СВЕДЕНИЯМИ О WINDOWS 95 
=========================================================
Кроме файла Readme.txt в состав Windows 95 входят и другие файлы с 
дополнительными сведениями.
	
	Config.txt	Синтаксис команд, помещаемых в файл Config.sys.

	Display.txt	Сведения о настройке драйверов видеоадаптера и 
			получении дополнительных драйверов. 

	Exchange.txt	Сведения об установке и работе 
			с Windows Messaging.

	Extra.txt	Сведения о размещении дополнительных файлов, 
			не входящих в состав Windows 95, например 
			обновленных драйверов.

	Faq.txt		Ответы на наиболее типичные вопросы, касающиеся 
			работы с Windows 95.

	General.txt	Сведения о потенциальных трудностях при загрузке 
			системы, программах, входящих в состав Windows 95, 
			средствах работы с дисками, драйверах, съемных 
			носителях, Microsoft FAX и поддержке рукописного 
			ввода.

			Этот файл содержит также данные, полученные 
			слишком поздно, чтобы включить их в другие файлы.
			Например, при наличии вопроса, относящегося к 
			принтерам, целесообразно заглянуть не только в файл 
			Printers.txt, но и в файл General.txt.
	
	Hardware.txt	Информация об известных проблемах при работе с 
			различным оборудованием и способах их устранения. 
			Часть подобных вопросов рассматривается также 
			в файлах Printers.txt и Mouse.txt.

	Infrared.txt	Сведения об установке и разрешении проблем 
                        при работе с инфракрасной связью.

	Internet.txt	Сведения о подключении к Internet,
			а также загрузке и использовании нового средства 
			Microsoft для просмотра содержимого Web-серверов.
			(Это средство называется проводником Internet.)

	Mailnews.txt	Сведения об установке, настройке
                        и разрешении проблем при работе 
			с Microsoft Internet Mail and News.

	Mouse.txt	Информация об известных проблемах при работе с
			клавиатурой и мышью и способах их устранения.

	Msdosdrv.txt	Синтаксис, использующийся при работе с 
			драйверами устройств MS-DOS. Дополнительные сведения
			по этому вопросу можно найти в файле Config.txt.
			Обратиться к встроенной справке любой из команд
			позволяет ключ /?, следующий за именем команды.

	Msn.txt		Сведения о подключении к Microsoft Network.

	Netmeet.txt	Сведения о программе Microsoft NetMeeting  и 
			об известных проблемах в при работе с ней.

	Network.txt	Сведения об особенностях работы в сети и
			использовании серверов сети.

	Printers.txt	Информация об известных проблемах при работе с
			принтерами и способах их устранения.

	Programs.txt	Информация об известных проблемах при работе с
			некоторыми программами для Windows и MS-DOS
			в Windows 95.

	Services.txt	Сведения о получении учетных записей
			пользователя у поставщиков услуг
			интерактивных информационных служб.

	Support.txt	Сведения о получении дополнительной поддержки, 
			относящейся к работе с Windows 95.

	Tips.txt	Сборник советов по работе с Windows 95. Большинство 
			этих советов не включено ни в документацию, ни в 
			справочную систему.
